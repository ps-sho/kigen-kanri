# 商品期限管理アプリ

商品の賞味期限・使用期限を店舗ごとに管理するウェブアプリ(iPhoneのホーム画面に追加して使用)。

## 構成(すべて無料)

```
iPhone → GitHub Pages(この画面) → GAS(受付係) → Googleスプレッドシート(データ)
                                              → Googleドライブ(写真)
```

| 役割 | サービス | 費用 |
|------|---------|------|
| 画面の配信 | GitHub Pages | 無料 |
| データの読み書き | Google Apps Script(ウェブアプリ) | 無料 |
| データ保存 | Googleスプレッドシート | 無料 |
| 写真保存 | Googleドライブ | 無料 |

- データはすべて自分のGoogleアカウント内に保存され、スプレッドシートで直接確認できます
- スプレッドシートの「**棚卸し**」シートに店舗ごとの在庫が自動集計されます
- 停止・課金の心配はありません(クレジットカード登録も不要)

## 主な機能

- 店舗別ログイン(他店舗の在庫・期限は閲覧のみ)
- バーコード読み取り(かざして読み取り/写真から読み取り/手入力)
- 商品登録: 写真・OCR商品名候補・期限(年月入力で月末扱いがデフォルト)・在庫数
- 期限警告: 期限切れ/7日以内/30日以内の色分けと絞り込み
- 在庫のワンタップ増減、別期限(新ロット)の追加、写真の後付け

## 接続設定

`index.html` 内の `GAS_API_URL` にGASウェブアプリのURL(～/exec)を設定します。
GAS側のコードとセットアップ手順は別途管理(リポジトリには含まれません)。

### 別のチームが自分のスプレッドシートで使う場合

このアプリは接続先を切り替えられます。自分のGoogleアカウントでGAS(データ係)を
デプロイし、アプリを一度だけ次のURLで開くと、以後その端末は自分のデータに接続されます:

```
https://ps-sho.github.io/kigen-kanri/?api=(自分のGASウェブアプリURL)
```
