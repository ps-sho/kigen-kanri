# 商品期限管理アプリ

商品の賞味期限・使用期限を店舗ごとに管理するPWA(iPhoneのホーム画面に追加して使えるウェブアプリ)。

## 主な機能

- **店舗別ログイン**: 6店舗それぞれのアカウントで利用。他店舗の在庫・期限は閲覧のみ可能
- **バーコード読み取り**: カメラでJANコード等を読み取り。登録済み商品は即座に表示、未登録なら登録画面へ
- **商品登録**: 商品名・写真・期限・在庫数を登録。写真からのOCR文字認識で商品名の入力候補を表示
- **期限警告**: 期限切れ/7日以内(赤)/30日以内(黄)を色分け表示・件数集計・絞り込み
- **在庫更新**: ワンタップで在庫の増減、新ロット(別期限)の追加
- **オフライン対応**: 通信がなくても閲覧・登録が可能。通信回復後に自動同期

## 構成(すべて無料)

| 役割 | サービス | 費用 |
|------|---------|------|
| アプリ配信 | GitHub Pages | 無料 |
| データベース・認証・写真保存 | Supabase(無料プラン) | 無料 |
| 停止防止ping | GitHub Actions(週1回) | 無料 |

- Supabase無料プランは一定期間アクセスがないと一時停止しますが、`.github/workflows/keepalive.yml` が週1回自動アクセスして防止します。
- コード内の `SUPABASE_KEY` は「公開してよい種類」のキーです。データ保護はサーバー側の行レベルセキュリティ(RLS)で行われ、店舗アカウントでログインしない限り読み書きできません。

## iPhoneでの初期設定(各店舗1回だけ)

1. Safariでアプリの公開URLを開く
2. 共有ボタン → **「ホーム画面に追加」**
3. ホーム画面の「期限管理」を開き、自店舗を選んでパスワードを入力(パスワードは『店舗パスワード一覧.md』参照 ※リポジトリには含まれません)
4. 初回にカメラの使用許可を求められたら「許可」

## ローカルでの動作確認

```bash
node serve.mjs   # → http://localhost:8123
```

## ファイル構成

```
index.html            画面
css/style.css         デザイン
js/config.js          Supabase接続設定
js/db.js              オフライン保存(IndexedDB)
js/app.js             アプリ本体
sw.js                 Service Worker(オフライン起動)
manifest.webmanifest  PWA設定
vendor/               ライブラリ(supabase-js, ZXing)
icons/                アプリアイコン
.github/workflows/    停止防止ping
```
